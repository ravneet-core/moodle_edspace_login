Local Plugin - edSpace Login
======================

Information
-----------

This plugin adds a new menu items at the end of Navigation Menu in order to allow user to log into edSpace.

Version
-------
Moodle 2.8+; 2.9+ , 3.0+, 3.1+ and 3.2+
